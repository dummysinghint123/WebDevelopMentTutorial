Properties of border
  • Border-width, To set the width of the border, Ex: border-width:15px
  • border-radius, To set the corners of the border rounded, , ex:border-radius:11px
  • border-style, to set the style of border, solid, dashed etc, Ex border-style:solid
  • border-color, to set the color of the border, ex border-color:red; to make the border of red color
  • in one line, border: width style color
  • we can have specific border border-top, border-left, border-bottom, border
  • we can have indiviual side properties for border also, Ex border=bottom-right-radius: 15px
